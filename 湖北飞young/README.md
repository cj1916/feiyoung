说明 以openwrt为例
===========================

## fy 联网脚本 将其放入/bin文件夹下，并增加权限
```
  cd /bin   chmod u+x fy
```
### *或用winscp直接修改权限

## fy.conf fy配置文件 将其放入/etc文件夹下，并增加权限
```
  cd /etc   chmod u+x fy.conf
```
